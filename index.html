SoC and CPU Floor Planning Project (OctoNyte)

Project Overview

The OctoNyte project focuses on System-on-Chip (SoC) floor planning and design automation using OpenLane and the SkyWater 130nm PDK.
My contributions include:

  • Designing functional partitions and validating floor plans
  • Integrating standard cell libraries and optimizing macro placements
  • Building and validating OpenLane 1.1.1 .tcl configuration files
  • Developing YAML configurations for OpenLane2 under a Nix-shell environment
