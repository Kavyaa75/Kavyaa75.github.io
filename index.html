SoC and CPU Floor Planning Project (OctoNyte)

Hardware Block Overview and Floor Planning Context

ALU32 (Arithmetic Logic Unit 32-bit)

Performs arithmetic (addition/subtraction) and logical operations. Clocked at 5ns. Utilization set to 30%, PDN pitch 6.0μm. SDC constraints applied. Layout completed under OpenLane 1.1.1.

LoadUnit

Manages memory read transactions to CPU registers. 5ns clock, 30% utilization, PDN pitch 7.0μm. TCL flow validated via DRC and LVS. Part of memory handling pipeline.

RISCVAdderSubtractor32

Executes 32-bit add/subtract operations for RISC-V instruction support. Used 10ns clock for relaxed timing. Defined in OpenLane with full constraints and verified SDC compliance.

TetraNyteCore

The untimed top-level RISC-V core integrating ALU, LoadUnit, and control paths. Initial logical integration point prior to synthesis. Verified for inter-module consistency.

TetraNyteCore_timed_rtl

Timed synthesizable version of TetraNyteCore. Built with integrated submodules (ALU32, LoadUnit). Used 5ns clock, 30% core utilization, PDN pitch 7.0μm. Fast DRC enabled in OpenLane.

SevenSegmentDisplay

Converts binary data to human-readable output using a seven-segment display. Peripheral logic block. Synthesized and validated using default constraints. Used for debugging interface.

mem_1r1w

Single-port synchronous memory allowing 1 read and 1 write. Used as base storage for register file and memory elements. Integrated with clocked logic and floorplanned for space optimization.

regfile_2r1w

Implements dual-read, single-write register file for CPU. Backed by mem_1r1w. Die area: 690μm × 230μm (158,700μm²). Manual pin placement via pin_order.cfg. Clock period: 10ns. YAML config executed in Nix-shell using OpenLane2.
